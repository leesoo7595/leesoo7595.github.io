### 백준 2839번

```python
n = int(input())

a = n // 3
b = n // 5

temp = []

for i in range(a):
    for j in range(b):
        if n == 3*a + 5*b:
            temp.append(a+b)

if len(temp) == 0:
    print(-1)
else:
    print(min(temp))
```

맞았긴 맞았는데 이중포문 드가서 그런지 속도가 너무 느리당

```python
n = int(input())
def test(n,b):
    if ((n-5*b)%3==0):
        return (n-5*b)//3+b
    elif ((n-5*b)%3!=0) & (b==0):
        return -1    
    else:
        return test(n,b-1)

b = n//5
print(test(n,b))
```

이건 친구가 옆에서 푼 방법인데 재귀로 풀었더라

요건 내꺼보다 세배나 빨랐당

```python
n=int(input())
print(-(n in[4,7])or n-2*n//5*2)
```
