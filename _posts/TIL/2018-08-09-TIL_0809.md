`django rest framework`에서 `serializer` 복습을 하다가 계속 나오는 `**kwargs`와 `*args` 개념이 머릿속에 잡혀있지않아 첫 TIL로 정리하였다.

어떤 값을 넣을진 알 수 없지만, `*args`는 함수에 변수가 튜플형태로 입력이 되어진다. <br>
그리고 `**kwargs`는 딕셔너리 형태로 입력되어진다고 보면 되겠다.

```python
def a(*args):
  print args

def b(**kwargs):
  print kwargs

a(1, 2, 3, 4, )
b(a=1, b=2, c=3)

print:
(1, 2, 3, 4)
{a:1, b:2, c:3}
```
